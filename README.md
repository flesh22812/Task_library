# task_library
Приложение системы ввода и отображения информации об авторах книг посредством REST  API (Java spring)
•	Автор
  ◦ Имя
  ◦	Фамилия
  ◦	Отчество
  ◦	Дата рождения
  ◦	Книги
•	Книга
  ◦	ISBN
  ◦	Жанр
•	Жанр
  ◦	Название
В приложении реализовано: CRUD операции, Liquibase database, WebSecurity auntification, Swagger,
